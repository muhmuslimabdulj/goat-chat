# Panduan Kontribusi GOAT Chat

Terima kasih telah tertarik untuk berkontribusi pada GOAT Chat! Kami sangat menghargai bantuan Anda untuk membuat proyek ini lebih baik.

## Cara Berkontribusi

### 1. Laporkan Bug
Jika Anda menemukan bug, silakan buat **Issue** di GitHub dengan format:
*   **Judul**: Deskripsi singkat bug.
*   **Deskripsi**: Langkah-langkah untuk mereproduksi bug.
*   **Ekspektasi**: Apa yang seharusnya terjadi.
*   **Realita**: Apa yang sebenarnya terjadi.
*   **Lingkungan**: Browser, OS, Tipe Device (Mobile/Desktop).

### 2. Request Fitur
Punya ide menarik? Buat **Issue** dengan label `enhancement` dan jelaskan ide Anda serta manfaatnya bagi pengguna lain.

### 3. Pull Request (PR)
Kami menerima Pull Request! Berikut langkah-langkahnya:

1.  **Fork** repository ini.
2.  Buat branch baru untuk fitur/fix Anda: `git checkout -b fitur-keren`.
3.  Commit perubahan Anda: `git commit -m 'Menambahkan fitur keren'`.
4.  Push ke branch Anda: `git push origin fitur-keren`.
5.  Buat **Pull Request** ke branch `main`.

## Standar Kode (Go)
*   Gunakan `gofmt` untuk memformat kode.
*   Pastikan tidak ada error saat menjalankan `go vet ./...`.
*   Usahakan kode modular dan mudah dibaca.
*   Hapus komentar debug atau kode mati sebelum commit.

## Code of Conduct
Harap bersikap sopan dan saling menghargai. Kami ingin komunitas yang inklusif dan ramah untuk semua orang.
